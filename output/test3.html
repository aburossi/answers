generate_html Function:

Retrieves the title and Markdown text from the GUI. Validates inputs. Reads the template.html file. Converts Markdown to HTML using the markdown library with extensions for extra features like code highlighting and table of contents. Replaces placeholders ({{title}} and {{content}}) in the template with the actual title and converted HTML content. Saves the generated HTML file in the output/ directory. Handles file overwriting by prompting the user. Displays success or error messages accordingly. GUI Elements:

Title Entry: A single-line input for the HTML page title. Markdown Textbox: A multi-line textbox for inputting Markdown content. Generate Button: Triggers the HTML generation process. Optional Output Directory Selection: (Commented out in the script but can be added if needed). Styling:

Basic styling for better user experience.
